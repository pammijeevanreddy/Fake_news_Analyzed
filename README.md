# 📰 Fake News Analysis & Misinformation Detection

## 📌 Project Overview

This project analyzes fake and real news articles to understand patterns in misinformation using data analysis, NLP techniques, SQL, and Power BI.

The goal is to identify how fake news differs from real news in terms of content, engagement, and behavior.

---

## 🛠️ Tools & Technologies

* Python (Pandas, NLP)
* SQL (MySQL)
* Power BI
* Excel

---

## 📂 Dataset

* Cleaned dataset containing news articles
* Includes features like:

  * title
  * text
  * type (fake, bias, real, etc.)
  * spam_score
  * likes, shares, comments
  * country, language

---

## 🔍 Key Steps

### 1. Data Cleaning

* Handled missing values
* Standardized date formats
* Removed irrelevant columns

### 2. Exploratory Data Analysis (EDA)

* Fake vs Real distribution
* Country-wise analysis
* Engagement trends
* Source analysis

### 3. NLP Analysis

* Text preprocessing (cleaning, lemmatization)
* Sentiment analysis
* Word frequency & n-grams
* Feature engineering (word count, title length)

### 4. SQL Analysis

* Aggregations and grouping
* Engagement comparisons
* Source-wise insights
* Time-based analysis

### 5. Power BI Dashboard

Created a 3-page interactive dashboard:

#### 📊 Page 1: Overview

* Fake vs Real distribution
* Articles over time
* Top countries and sources

#### 📈 Page 2: Engagement Analysis

* Likes, shares, comments comparison
* Viral content analysis
* Spam score distribution

#### 🧠 Page 3: NLP Insights

* Word count analysis
* Title behavior
* Language distribution
* Source credibility

## 🔑 Key Insights

* Fake news tends to use emotionally charged language
* Fake articles show higher engagement (shares, likes)
* Certain sources consistently produce misleading content
* Spam score correlates with virality
